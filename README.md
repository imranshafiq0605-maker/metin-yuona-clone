# Metin2 fantasy landing page starter

## Install

```bash
npm install
npm run dev
```

## Image placeholders

Put your images here:

- `public/assets/backgrounds/main-bg.jpg`
- `public/assets/hero/hero-character.png`
- `public/assets/logo/logo-main.png`
- `public/assets/logo/logo-footer.png`
- `public/assets/icons/stat-icon.png`
- `public/assets/icons/horns-top.png`
- `public/assets/icons/class-icon.png`
- `public/assets/news/video-cover.jpg`
- `public/assets/news/news-thumb-01.jpg`
- `public/assets/slider/center-character.png`
- `public/assets/banners/discord-banner-01.jpg`

If you want different banner images, duplicate the CSS blocks or update the JSX structure.

## Deploy on Vercel

1. Push this project to GitHub.
2. Import the repository in Vercel.
3. Framework preset: Next.js.
4. Build command: `next build`
5. Output setting: default.
